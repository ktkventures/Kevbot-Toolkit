## Dashboard Components

### 1. Welcome Header
- User greeting
- Current date and market status

### 2. Quick Actions
- Large, clickable cards for common tasks
- New Strategy, View Strategies, New Portfolio, View Alerts

### 3. Portfolio Summary
- Count of active strategies/portfolios
- P&L breakdown (today, week, month)
- Dollar values based on P&L settings

### 4. Forward Test Overview
- Status breakdown of all strategies
- Highlight best and worst performers
- Quick links to strategies needing attention

### 5. Recent Alerts
- Timeline of recent entry/exit signals
- Quick context on each alert
- Link to full alert history

### 6. Combined Equity Curve
- Aggregated performance across all active strategies
- Shows total account growth
- Date range selector (optional)
