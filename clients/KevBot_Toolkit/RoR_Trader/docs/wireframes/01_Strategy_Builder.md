# RoR Trader - Strategy Builder Wireframes

**Priority:** P0 (Core Feature)
**Inspired by:** Trade Analyzer drill-down interface

---

## Overview

The Strategy Builder is a 3-step wizard that guides users through creating a trading strategy:
1. **Setup** - Define ticker, direction, and triggers
2. **Confluence** - Layer in confluence conditions (drill-down)
3. **Save** - Review and save the strategy

The layout mirrors the Trade Analyzer, with the key addition of a step indicator and more prominent equity curve.

### Step 3 Interactions
- **Strategy Name**: Auto-generated suggestion, user can edit
- **Summary**: Read-only review of all selections
- **Backtest Results**: Final KPIs from confluence selection
- **Forward Testing Toggle**: Enables live tracking after save
- **Alerts Toggle**: Configure notifications (can set up later in detail)
- **Save**: Creates strategy, redirects to Strategy Detail page

## Mobile Considerations

On mobile, Step 2 would stack:
1. Strategy header + KPIs (always visible at top)
2. Active confluence tags (horizontal scroll)
3. Equity curve (full width, shorter height)
4. Confluence panel (full width, scrollable)
5. Trade list (collapsed by default)

Navigation buttons become fixed bottom bar.
